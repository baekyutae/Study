# 풀이 아이디어

화폐를 오름 차순으로 정렬

불변식(가정): 지금까지 처리한 동전들로 1부터 target-1까지는 전부 만들 수 있고, target은 만들 수 없는 최소 금액 후보이다. (target은 “못 만드는 최소값”)

다음 동전이 x일 때

x > target이면, 새 동전을 쓰면 최소 합이 x라서 target을 만들 수 없고, 정렬 때문에 이후 동전도 더 커서 영원히 target을 만들 수 없다 → 정답은 target.

x <= target이면, 기존에 만들 수 있던 1~target-1에 x를 더해서 x~x+target-1도 만들 수 있다. x<=target이라 두 구간이 이어져서 1~(target+x-1)까지 전부 만들 수 있게 되므로, 새로운 “못 만드는 최소 금액”은 target+x가 된다 → target += x.

# 테스트 케이스

cases = [
    # n, 동전 리스트, 예상결과

    # 1. 최소 케이스: 1원만 있음
    (1, [1], 2),

    # 2. 정렬 안된 입력
    (3, [3, 2, 1], 7),

    # 3. 기본예시
    (5, [3, 2, 1, 1, 9], 8),

    # 4. 전부 1원(중복 다수)
    (6, [1, 1, 1, 1, 1, 1], 7),

    # 5. 중복 + 큰 점프
    (5, [1, 2, 2, 2, 10], 8),

    # 6. 중간에 빈 구간이 생김
    (3, [1, 2, 5], 4),

    # 7. 첫 동전이 1보다 큼(즉시 1)
    (4, [2, 3, 4, 5], 1),

]
